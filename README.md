# Trader Sentiment Analysis

## Overview

This project analyzes how market sentiment influences trader behavior. The analysis combines historical trader data with market sentiment data to identify patterns in trading activity, leverage usage, position sizes, and trading performance.

## Objectives

- Analyze trader behavior under different market sentiments.
- Compare trading frequency across sentiment categories.
- Study leverage usage and position sizes.
- Analyze long vs. short trading bias.
- Segment traders based on trading characteristics.
- Generate insights using data visualization.

## Dataset

The project uses two datasets:

1. Historical Trader Data
2. Market Sentiment Data (Fear, Neutral, Greed)

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed

- Data cleaning and preprocessing
- Merging trader and sentiment datasets
- Exploratory Data Analysis (EDA)
- Trade frequency analysis
- Leverage analysis
- Position size analysis
- Long vs. Short bias analysis
- Trader segmentation
- Visualization of key insights

## Key Insights

- Trader activity changes across different market sentiment categories.
- High-leverage traders exhibit different risk-taking behavior than low-leverage traders.
- Trading frequency varies with market sentiment.
- Position sizes and trading direction show noticeable patterns during Fear and Greed periods.
